# Sistema de Gestión Climática

## Descripción

Sistema desarrollado en Node.js para gestionar registros climáticos.

## Funcionalidades

- Crear registros
- Consultar registros
- Editar registros
- Eliminar registros
- Dashboard
- Alertas climáticas

## Tecnologías

- Node.js
- JavaScript
- HTML
- CSS

## Ejecución

npm start
